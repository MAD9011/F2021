MAD9011

Mobile Design Fundamentals

Get Started →

Basic design concepts and vector-based graphics

Learn design fundamentals. Understand different graphics formats. Create vector-based graphics in Adobe Illustrator.

Layout concepts and raster-based graphics

Understand layout principles. Use Photoshop to create raster-based graphics and layout. Optimize graphics for mobile user interface.

SVG graphics & markups and user interface design

Create scalable SVG graphics and markups. Design mobile user interface with Adobe XD. Create interactive visual mockups with all tools learned.
